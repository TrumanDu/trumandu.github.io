Nginx Docker构建https服务

发表于 2020-09-23 更新于 2021-07-05 分类于 nginx

前言

本篇文章介绍生成一个自签名SSL证书以及使用Nginx docker 代理一个https服务。

SSL证书验证安全连接，有两种验证模式：

  1. 仅客户端验证服务器的证书，客户端自己不提供证书；
  2. 客户端和服务器都互相验证对方的证书。

显然第二种更安全，一般web采用第一种，比较简单。
